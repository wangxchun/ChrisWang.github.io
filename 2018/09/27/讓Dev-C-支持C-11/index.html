讓 Dev C++ 支持 C++11

發表於 2018-09-27 更新於 2021-04-01 分類於 C++學習

在 DevC++ 中


會出現如下錯誤

解決方法

(1) 先在 dev 的 [工具] 里找到 [編譯選項]
(2) 勾選 [編譯時加入以下命令] 欄位並填入 - std=c++11

# Dev C++ # C++ 11
