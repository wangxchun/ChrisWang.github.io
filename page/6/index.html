發表於 2021-05-10 更新於 2021-05-17 分類於 生活-投資理財

一、基本面三大方向，九大指標

1. 做什麽 (公司狀況)：

(1) 資本額 (股本)
股本就是公司設立時的資本額 = 股數 X 股票面額 (10 元)
市值 = 股票數 X 市價 (面額)(成交價)
(2) 市值
(3) 產業

2. 賺什麽 (經營績效)

(1) EPS = 稅後净利 / 股數
盈餘分配率
(2) ROA 資產報酬率 = 稅後净利 / 資產
(3) ROE 股東權益報酬率 = 稅後净利 / 净值

3. 算什麽 (市況指標)

遛狗理論
(1) 本益比 PER（賺多少）：股價 / EPS，同業比較還要注意經營内容的差異

  • 跟自己比 - 和同業比
    (2) 股價净值比 PBR（值多少）：目前股價 / 每股净值
    每股净值 = 净值 / 股數
  • 净值的代表性 - 沒有評估獲利能力
    (3) 殖利率 Yield（配多少）：
    利率 / 股價

二、巴菲特的投資秘訣

1. 投資兩個要素：

(1) 價值被低估
(2) 長期持有

2. 怎麽看一家公司是否被低估

(1) 管理層：企業所有者和競爭優勢
(2) 經營數據
